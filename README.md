# 一个可以放置在桌面的便签
+ 窗口可置顶、置底、锁定位置和大小、隐藏、透明度等
+ 窗口可以鼠标穿透
+ 储存方式为：转码base64储存
  # 使用方式
  JavaVersion:1.8
  启动运行NoteUI.java
# Tag -v1.0.0
  下载exe，点击后即可使用，数据储存在C:\\Users\\<username>\\AppData\\Local\\fengxue\\note
  若有其他需求可自行拉取代码修改
