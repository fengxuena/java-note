# 一个可以放置在桌面的便签
+ 窗口可置顶、置底、锁定位置和大小、隐藏等
+ 窗口可以鼠标穿透
+ 储存方式转码base64储存，在appdata目录内
  # 使用方式
  启动main.java
# Tag -v1.0.0
  下载exe，点击后即可使用，数据储存在C:\Users\<username>\AppData\Local\fengxue\note下
