# 一个可以放置在桌面的便签
+ 窗口可置顶、置底、锁定位置和大小、隐藏、透明度等
+ 窗口可以鼠标穿透
+ 储存方式为：转码base64储存
+ 右键标签名称可以删除当前标签和数据
+ 左键点击“添加标签”可以添加新标签
+ 数据储存有两种方式，窗口失去焦点、正常退出程序
+ 
# Tag -v1.0.1

+ 添加鼠标穿透热键
+ 使用可编辑的css
